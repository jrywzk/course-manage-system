# 成绩字段补全说明 (2026-05-15)

> **修改人**：成员 D
> **影响范围**：EnrollmentVo 新增 `usualScore`（平时成绩）、`examScore`（考试成绩）

---

## 一、修改内容

`EnrollmentVo` 在原有字段基础上新增两个字段：

| 字段 | 类型 | 说明 |
|------|------|------|
| `usualScore` | BigDecimal | 平时成绩（有成绩时返回，无成绩时为 null） |
| `examScore` | BigDecimal | 考试成绩（有成绩时返回，无成绩时为 null） |

原有成绩相关字段不变：`hasScore`（是否有成绩）、`finalScore`（综合成绩）、`gpaPoint`（GPA绩点）、`isPassed`（是否通过）。

---

## 二、受影响接口

### 1. GET /api/students/{studentId}/enrollments — 我的课程

返回类型：`List<EnrollmentVo>`

新增响应字段示例：
```json
{
  "data": [{
    "enrollmentId": 19,
    "sectionId": 1,
    "courseName": "程序设计基础",
    "teacherName": "王教授",
    "hasScore": true,
    "usualScore": 85.00,
    "examScore": 82.00,
    "finalScore": 83.20,
    "gpaPoint": 3.3,
    "isPassed": 1
  }]
}
```

### 2. GET /api/sections/{sectionId}/students — 教学班学生名单

返回类型：`List<Map<String,Object>>`

新增响应字段示例：
```json
{
  "data": [{
    "enrollmentId": 19,
    "studentId": 8,
    "studentNo": "S2025008",
    "studentName": "冯小文",
    "gender": "M",
    "majorName": "计算机科学与技术",
    "selectTime": "2025-09-03 09:00:00",
    "hasScore": true,
    "usualScore": 85.00,
    "examScore": 82.00,
    "finalScore": 82.90,
    "isPassed": 1
  }]
}
```

## 三、完整字段列表（EnrollmentVo）

| 字段 | 类型 | 说明 |
|------|------|------|
| enrollmentId | Integer | 选课记录ID |
| sectionId | Integer | 教学班ID |
| sectionCode | String | 教学班编码 |
| semester | String | 学期 |
| courseCode | String | 课程编码 |
| courseName | String | 课程名称 |
| credit | BigDecimal | 学分 |
| teacherName | String | 授课教师 |
| scheduleText | String | 上课时间 |
| building | String | 教学楼 |
| roomNo | String | 教室号 |
| selectTime | String | 选课时间 |
| status | Integer | 选课状态 1=在读 0=已退课 |
| hasScore | Boolean | 是否已有成绩 |
| **usualScore** | **BigDecimal** | **平时成绩（新增）** |
| **examScore** | **BigDecimal** | **考试成绩（新增）** |
| finalScore | BigDecimal | 综合成绩 |
| gpaPoint | BigDecimal | GPA绩点 |
| isPassed | Integer | 是否通过 |
